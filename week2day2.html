Data manipulation is a core step in data analytics where raw datasets are transformed into a meaningful and analysis-ready format. In R, the dplyr package provides a consistent and intuitive set of functions for performing common data manipulation tasks efficiently.

This session focuses on row-wise and column-wise operations such as filtering records, selecting relevant variables, sorting observations, and creating new derived columns. These operations are essential for cleaning datasets, preparing features, and simplifying complex data analysis workflows.

1. Introduction to dplyr

The dplyr package is part of the tidyverse and is designed for fast and readable data manipulation. It works primarily with data frames and tibbles. 

library(dplyr)

Concepts Covered:

  • Tidy data philosophy
  • Pipe-friendly data manipulation

    • 2. Filtering Rows using filter()

    The filter() function is used to extract rows that satisfy given logical conditions. 

    filter(df, Age > 18)
filter(df, Department == "IT")

    Concepts Covered:

  • Row-wise filtering
  • Logical conditions

    • 3. Selecting Columns using select()

    The select() function allows choosing specific columns from a dataset, helping reduce clutter and focus on relevant variables. 

    select(df, Name, Age)
select(df, starts_with("Sal"))

    Concepts Covered:

  • Column selection
  • Helper functions

    • 4. Sorting Data using arrange()

    The arrange() function is used to sort rows in ascending or descending order based on one or more variables. 

    arrange(df, Age)
arrange(df, desc(Salary))

    Concepts Covered:

  • Ascending and descending sorting
  • Multi-column sorting

    • 5. Creating New Columns using mutate()

    The mutate() function creates new variables or modifies existing ones using calculations based on other columns. 

    mutate(df, Bonus = Salary * 0.10)
mutate(df, AgeSquared = Age^2)

    Concepts Covered:

  • Feature engineering
  • Derived variables

    • CODING PRACTICE

    library(dplyr)

employees <- data.frame(
  Name = c("Amit", "Neha", "Ravi", "Simran"),
  Age = c(23, 29, 21, 25),
  Department = c("IT", "HR", "IT", "Finance"),
  Salary = c(40000, 50000, 35000, 45000)
)

# Filter employees older than 22
filter(employees, Age > 22)

# Select relevant columns
select(employees, Name, Department)

# Sort by salary
arrange(employees, desc(Salary))

# Create new column
mutate(employees, AnnualSalary = Salary * 12)
