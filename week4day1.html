3D Plotting, Animation & Histograms in R

Concepts Covered:
3D plotting using rgl, animated bar plots using gganimate, base R histograms and bar plots, multiple histograms, advanced histograms using ggplot2

1. 3D Plotting Using rgl

The rgl package enables interactive 3D plotting, rotation, zooming, and real-time visualization for multivariate data.

library(rgl)

# Sample Data
x <- c(1, 2, 3, 4, 5)
y <- c(2, 5, 3, 6, 4)
z <- c(4, 7, 5, 8, 6)

# 3D Scatter Plot
plot3d(x, y, z, col = "blue", size = 5, 
       xlab = "X axis", ylab = "Y axis", zlab = "Z axis")

# 3D Lines
lines3d(x, y, z, col = "red", lwd = 3)

# Add bounding box and background
bg3d(color = "lightgrey")
bbox3d(color = c("black", "green", "blue"))

# 3D Animation
play3d(spin3d(axis = c(0, 0, 1), rpm = 5), duration = 10)

Additional Features: points3d(), text3d(), segments3d(), and surface3d() for complex shapes or meshes.

2. Animated Bar Plots Using gganimate

gganimate allows creating animations for time-varying data. Ideal for monthly trends, stock data, or category comparisons.

library(ggplot2)
library(gganimate)
library(gifski)

# Sample Data
data <- data.frame(
  Month = rep(c("Jan", "Feb", "Mar"), each = 3),
  Category = rep(c("Rent", "Food", "Transport"), times = 3),
  Amount = c(1200, 800, 300, 1250, 850, 350, 1300, 900, 400)
)

# Animated Bar Plot
p <- ggplot(data, aes(x = Category, y = Amount, fill = Category)) +
  geom_bar(stat = "identity") +
  labs(title = "Month: {closest_state}", x = "Category", y = "Amount") +
  theme_minimal() +
  transition_states(Month, transition_length = 1, state_length = 1)

animate(p, renderer = gifski_renderer())
anim_save("monthly_expenses.gif", animation = last_animation())

3. Base R Histograms & Bar Plots

Base R provides simple ways to examine distributions and compare groups without external packages.

# Histogram Example
values <- c(12, 15, 18, 21, 25, 30, 35, 40, 45, 50)
hist(values, breaks = 5, col = "skyblue", border = "black",
     main = "Histogram Example", xlab = "Values", ylab = "Frequency")

# Bar Plot Example
categories <- c("A", "B", "C", "D")
counts <- c(10, 15, 7, 20)
barplot(counts, names.arg = categories, col = "orange", 
        main = "Bar Plot Example", ylab = "Count", xlab = "Category")

4. Multiple Histograms in One Frame

Use par(mfrow = c(rows, cols)) to display multiple plots simultaneously.

par(mfrow = c(2, 2), bg = "lightyellow")

hist(iris$Sepal.Length, main = "Sepal Length", xlab = "Length", col = "skyblue")
hist(iris$Sepal.Width, main = "Sepal Width", xlab = "Width", col = "orange")
hist(iris$Petal.Length, main = "Petal Length", xlab = "Length", col = "lightgreen")
hist(iris$Petal.Width, main = "Petal Width", xlab = "Width", col = "plum")

5. Histograms Using ggplot2

ggplot2 allows layered, customized, and color-coded histograms. Ideal for reports and dashboards.

library(ggplot2)

# Single Histogram
ggplot(iris, aes(x = Sepal.Length)) +
  geom_histogram(binwidth = 0.3, fill = "skyblue", color = "black") +
  labs(title = "Sepal Length Distribution", x = "Sepal Length", y = "Count") +
  theme_minimal()

# Overlay Histograms by Species
ggplot(iris, aes(x = Sepal.Length, fill = Species)) +
  geom_histogram(binwidth = 0.3, position = "identity", alpha = 0.5) +
  labs(title = "Sepal Length by Species", x = "Sepal Length", y = "Count") +
  theme_minimal()

6. Additional Tips

  • For interactive 3D plots, try zoom3d(), rotate3d(), and rglwidget() for Shiny integration.
  • Use alpha blending in ggplot2 histograms to show overlapping distributions clearly.
  • Combine multiple animations using transition_manual() for stepwise comparison.
