Statistical Analysis

Statistical analysis involves collecting, organizing, summarizing, and interpreting data to uncover patterns and trends. It helps in making informed decisions by applying mathematical techniques such as measures of central tendency (mean, median, mode), dispersion (variance, standard deviation), and position (quantiles, percentiles).

Input Vector:

myvector <- c(23, 43, 78, 21, 12, 32, 43, 54, 65, 16)
print(myvector)
sum(myvector)
min(myvector)
max(myvector)
sort(myvector)
which.max(myvector)
which.min(myvector)
mean(myvector)
median(myvector)
var(myvector)
quantile(myvector, 0.25)
names(sort(-table(myvector)))
mode(myvector)

Statistical Tests in R

vect1 <- c(23, 34, 21, 12, 21, 34, 56, 62, 22, 18)
vect2 <- c(34, 21, 43, 23)

1. T-Test

t.test(vect1, vect2)

2. Chi-Square Test

chisq.test(vect1)

3. F-Test

var.test(vect1, vect2)

4. Correlation Test

cor.test(vect1[1:4], vect2)
