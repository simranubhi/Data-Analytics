Week 4 - Day 2

Advanced Data Visualization in R: ggplot2 & plotly

Concepts Covered:
Layered plots using ggplot2, scatter plots, line plots, boxplots, interactive plots using plotly, 3D interactive plots, combining multiple plots

1. ggplot2 Basics

ggplot2 uses the Grammar of Graphics for building complex plots layer by layer. You can add points, lines, smoothers, labels, and themes incrementally.

library(ggplot2)

# Scatter Plot
ggplot(iris, aes(x = Sepal.Length, y = Petal.Length, color = Species)) +
  geom_point(size = 3) +
  labs(title = "Sepal vs Petal Length", x = "Sepal Length", y = "Petal Length") +
  theme_minimal()

# Line Plot
ggplot(economics, aes(x = date, y = unemploy)) +
  geom_line(color = "darkred", size = 1) +
  labs(title = "Unemployment Over Time", x = "Date", y = "Number of Unemployed") +
  theme_light()

2. Boxplots and Violin Plots

Used to visualize distribution and spread of data, including outliers.

# Boxplot
ggplot(iris, aes(x = Species, y = Sepal.Length, fill = Species)) +
  geom_boxplot() +
  labs(title = "Sepal Length Distribution by Species") +
  theme_minimal()

# Violin plot
ggplot(iris, aes(x = Species, y = Sepal.Length, fill = Species)) +
  geom_violin(trim = FALSE) +
  labs(title = "Violin Plot of Sepal Length") +
  theme_minimal()

3. Interactive Plots Using plotly

plotly converts ggplot2 plots into interactive plots with zoom, pan, hover info.

library(plotly)

# Convert ggplot to interactive
p <- ggplot(iris, aes(x = Sepal.Length, y = Petal.Length, color = Species)) +
  geom_point(size = 3)
ggplotly(p)

# 3D Scatter Plot
plot_ly(iris, x = ~Sepal.Length, y = ~Sepal.Width, z = ~Petal.Length,
        color = ~Species, size = ~Petal.Width, type = "scatter3d", mode = "markers")

4. Combining Multiple Plots

Use patchwork or gridExtra to combine multiple ggplot plots.

library(gridExtra)

p1 <- ggplot(iris, aes(Sepal.Length, Petal.Length, color = Species)) + geom_point()
p2 <- ggplot(iris, aes(Sepal.Width, Petal.Width, color = Species)) + geom_point()

grid.arrange(p1, p2, ncol = 2)
