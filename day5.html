DATA IMPORT AND CLEANING BASICS

Reading and Writing CSV Files

In R, the most common format for importing and exporting datasets is the CSV (Comma-Separated Values) file. The functions read.csv() and write.csv() are used for reading and writing CSV files respectively. 

# Reading a CSV file
data <- read.csv("students.csv")

# Writing to a CSV file
write.csv(data, "output_students.csv", row.names = FALSE)

Reading Excel Files

To read Excel files, use the readxl or openxlsx packages. These allow importing data from multiple sheets or writing Excel files easily. 

library(readxl)
excel_data <- read_excel("data_file.xlsx", sheet = "Sheet1")

library(openxlsx)
write.xlsx(excel_data, "output_data.xlsx")

Reading and Writing Text Files

Text files can be read using read.table() and written using write.table(). The delimiter and header options can be customized based on the file type. 

text_data <- read.table("data.txt", header = TRUE, sep = "\t")
write.table(text_data, "output.txt", sep = "\t", row.names = FALSE)

Data Exploration Basics

Once data is imported, it is essential to explore its structure, dimensions, and summary statistics before analysis. 

str(data)        # Structure of dataset
summary(data)    # Statistical summary
head(data)       # First few rows
tail(data)       # Last few rows
ncol(data)       # Number of columns
nrow(data)       # Number of rows

Handling Missing Values

Missing values in R are represented by NA. They can be detected, replaced, or removed depending on the situation. 

# Detect missing values
sum(is.na(data))

# Remove rows with missing values
clean_data <- na.omit(data)

# Replace missing values with mean of the column
data$Marks[is.na(data$Marks)] <- mean(data$Marks, na.rm = TRUE)

Filtering and Selecting Columns

The dplyr package provides easy and readable syntax for filtering and selecting data. 

library(dplyr)

# Filter rows where Marks > 50
filtered <- filter(data, Marks > 50)

# Select only specific columns
selected <- select(data, Name, Marks)

Renaming Columns

Columns can be renamed using rename() from the dplyr package. 

data <- rename(data, StudentName = Name, Score = Marks)

Type Conversion

You can convert column data types using as.numeric(), as.character(), or as.factor(). 

data$RollNo <- as.numeric(data$RollNo)
data$Class <- as.character(data$Class)
data$Grade <- as.factor(data$Grade)

String Handling

String manipulation is commonly performed using the stringr package. It provides functions for pattern matching and modification. 

library(stringr)

# Convert text to upper/lower case
data$Name <- str_to_upper(data$Name)
data$City <- str_to_title(data$City)

# Detect and replace patterns
data$City <- str_replace(data$City, "Delhi", "New Delhi")

# Count characters in strings
str_length(data$Name)

Exporting Cleaned Data

After cleaning and processing, the dataset can be saved as a new CSV or Excel file for further use. 

write.csv(clean_data, "cleaned_students.csv", row.names = FALSE)
write.xlsx(clean_data, "cleaned_students.xlsx")
