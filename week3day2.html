Week 3 (DAY 2)

Correlation & Hypothesis Testing

Concepts Covered:
Correlation Analysis, Pearson & Spearman correlation, Hypothesis testing, t-test, Chi-square test, p-value interpretation

1. Correlation Analysis

  • Measures the strength and direction of the relationship between two numeric variables.
  • Correlation coefficient ranges from -1 to +1.
  • Positive correlation: variables increase together.
  • Negative correlation: one variable increases while the other decreases.
  • Zero correlation: no linear relationship.

Example in R:

# Sample vectors
x <- c(10, 20, 30, 40, 50)
y <- c(15, 25, 35, 45, 55)

# Pearson correlation
cor(x, y, method = "pearson")

# Spearman correlation
cor(x, y, method = "spearman")

2. Hypothesis Testing

  • Hypothesis testing checks whether a claim about a population is true using sample data.
  • Null Hypothesis (H0): Assumes no effect or no difference.
  • Alternative Hypothesis (H1): Assumes there is an effect or difference.
  • Decisions are based on p-value and significance level (α = 0.05).

Common Tests:

  • t-test: Compares means of two groups.
  • Chi-square test: Tests association between categorical variables.
  • F-test: Compares variances of two samples.

t-test Example in R:

group1 <- c(23, 34, 21, 45, 33)
group2 <- c(30, 40, 28, 50, 36)

# Two-sample t-test
t.test(group1, group2)

Chi-square Test Example in R:

# Observed frequencies
observed <- c(20, 30, 50)

# Chi-square goodness of fit test
chisq.test(observed)

Interpreting p-values

  • If p-value < α (0.05), reject H0 → evidence supports H1.
  • If p-value > α, fail to reject H0 → not enough evidence to support H1.

Engineering Applications

  • Correlation between temperature and material expansion.
  • Hypothesis testing to validate machine calibration.
  • Comparing efficiency of two pump models using t-tests.
