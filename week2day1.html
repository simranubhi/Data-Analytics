Data analysis begins with understanding the structure and nature of data. In R, data is commonly stored in structured formats such as vectors, matrices, and data frames. Working effectively with data requires the ability to explore, summarize, and interpret datasets before applying advanced transformations or visualizations.

R provides a wide range of built-in datasets that are extremely useful for learning data exploration and basic analysis techniques. Using the data() function, we explored datasets available across installed packages and identified commonly used datasets from the base datasets package. Two important datasets — AirPassengers and iris — were studied in detail. The AirPassengers dataset introduced time-series data, while the iris dataset helped understand structured tabular data. This session focused on inspecting datasets, understanding their structure, and computing basic descriptive statistics, forming a foundation for data manipulation in subsequent sessions.

1. Accessing Built-in Datasets in R

R includes several built-in datasets that allow users to practice data exploration and analysis without importing external files.

data()
data(package = .packages(all.available = TRUE))
all.datasets <- data(package = "datasets")$results[, "Item"]
length(all.datasets)

Concepts Covered:

  • Viewing all available datasets
  • Counting the number of built-in datasets
  • Loading datasets for analysis

    • 2. Exploring the AirPassengers Dataset (Time Series Data)

    The AirPassengers dataset represents monthly international airline passenger counts from 1949 to 1960 and is widely used to introduce time-series data concepts. 

    ?AirPassengers
abc = AirPassengers
dim(AirPassengers)

    Concepts Covered:

  • Accessing R documentation using ?
  • Assigning datasets to variables
  • Checking dataset dimensions

    • 3. Exploring the iris Dataset (Tabular Data)

    The iris dataset contains measurements of iris flowers across three species and is commonly used for learning data exploration techniques using data frames. 

    ?iris
abc = iris
dim(abc)
names(abc)
rownames(abc)

    Concepts Covered:

  • Understanding rows and columns
  • Exploring dataset structure
  • Identifying column and row names

    • 4. Accessing Columns and Exploring Values

    Individual columns of a data frame can be accessed using the $ operator, allowing focused analysis of specific variables. 

    abc$Sepal.Length
sort(abc$Sepal.Length)
summary(abc)

    Concepts Covered:

  • Extracting columns using the $ operator
  • Sorting numeric values
  • Generating summary statistics

    • 5. Descriptive Statistics

    Descriptive statistics help summarize numeric data using measures such as minimum, maximum, mean, and median. 

    min(abc$Sepal.Length)
max(abc$Sepal.Length)
which.max(abc$Sepal.Length)
which.min(abc$Sepal.Length)
mean(abc$Sepal.Length)
median(abc$Sepal.Length)

    Concepts Covered:

  • Identifying minimum and maximum values
  • Finding index positions of extreme values
  • Calculating mean and median

    • 6. Finding the Mode

    R does not provide a direct function to compute the statistical mode for numeric data. The mode can be derived using frequency tables. 

    names(sort(-table(abc$Sepal.Length)))

    Concepts Covered:

  • Creating frequency tables
  • Sorting frequencies
  • Identifying the most frequent value
