Week 4 - Day 3

Statistical Analysis & Hypothesis Testing in R

Concepts Covered:
Descriptive statistics, mean, median, variance, standard deviation, t-tests, chi-square tests, correlation, ANOVA

1. Descriptive Statistics

summary(iris)
mean(iris$Sepal.Length)
sd(iris$Sepal.Length)
var(iris$Sepal.Length)
quantile(iris$Sepal.Length, probs = c(0.25, 0.5, 0.75))

2. Correlation Analysis

cor(iris$Sepal.Length, iris$Petal.Length)
cor(iris[,1:4])  # Pairwise correlation for all numeric columns

3. T-Test

# Compare Sepal.Length between two species
t.test(Sepal.Length ~ Species, data = subset(iris, Species != "virginica"))

4. ANOVA

anova_result <- aov(Sepal.Length ~ Species, data = iris)
summary(anova_result)

5. Chi-Square Test

table_data <- table(iris$Species, iris$Sepal.Width > 3)
chisq.test(table_data)
